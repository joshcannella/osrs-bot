# BarbFishCookScript — Setup Instructions

## RuneLite Requirements (Mandatory)
- Windows Display Scaling: **100%**
- RuneScape UI: **"Fixed - Classic"**
- Display Brightness: **middle (50%)**
- ChromaScape RuneLite Profile: **activated**

## RuneLite Plugin Configuration

### Cyan Colour Tag
- Tag the target in **cyan** — HSV ~90, 254, 254
- Use **Hull** or **Tile** highlight style

### Red Colour Tag
- Tag the target in **red** — HSV ~0, 254, 254
- Use **Hull** or **Tile** highlight style

### Green Colour Tag
- Tag the target in **green** — HSV ~60, 254, 254
- Use **Hull** or **Tile** highlight style

### Black Colour Tag
- Tag the target in **black** — HSV ~0, 0, 0
- Use **Hull** or **Tile** highlight style

### Idle Notifier
- **Enable** this plugin — required for idle detection

## Game Settings
- **Shift-click drop**: must be enabled (Settings → Controls)

## How to Run
- Script class: **BarbFishCookScript**
- Start ChromaScape, open the web UI at `http://localhost:8080/`
- Select **BarbFishCookScript** from the sidebar and click Start
