ChromaScape requires specific client configurations to ensure optimal performance and reliability. Please follow these requirements carefully before running any scripts.

## Windows Scaling
Windows 10/11 – Set Display Scaling to 100%
* Right-click on your desktop and select Display settings.
* Under Scale and layout, find Change the size of text, apps, and other items.
* From the drop-down menu, select 100%.

## RuneScape Settings UI Layout
You should set your UI to be either "Resizable - Classic" or "Fixed - Classic". "Resizable - Modern" will unfortunately not be supported due to template dependencies.

## Display Brightness Configuration
Purpose: Ensures consistent colour detection across different lighting conditions and monitors.

### Required Setting:
Set the in-game brightness slider to the middle position (50%)
This standardizes colour values for template matching and HSV detection algorithms

> Important: Deviating from this brightness setting may cause colour-based detection methods to fail or behave inconsistently.

## ChromaScape RuneLite settings:
- Start the program see the [Making Your First Script](./Making-your-first-script) page for instructions.
- Ensure that the Web UI has loaded.
- Click the wrench in the top right sidebar in RuneLite.
- Click the RuneLite Profiles icon, with the image of two people.
- Select the new `ChromaScape` profile by double clicking it.
> This will standardise your plugins to work with the project.
