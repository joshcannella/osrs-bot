# OSRS New Player Guide

Source: https://oldschool.runescape.wiki/w/New_player_guide

## Key Facts

- RuneScape login credentials work on OSRS but previous progress does not carry over
- Account progression is NOT shared between OSRS and RS3
- Membership status IS shared between OSRS and RS3
- Progression is not tied to a specific world/server — all worlds share progress
- Can play on PC or mobile with shared progression
- Only one login location at a time per account

## Tutorial Island

- All new accounts start here
- Teaches basic controls and gameplay
- Players customize character appearance (can change later via makeover)
- Must talk to Gielinor Guide — telling him you're new enables Adventure Paths
- Paul on Tutorial Island allows setting Ironman restrictions

### Ironman Modes

- **Ironman**: No trading with other players
- **Ultimate Ironman**: No trading, no bank storage
- **Hardcore Ironman**: Regular ironman with 1 life; death reverts to regular ironman
- **Group Ironman**: Trade only within group (2-5 players); can start as hardcore
- Adam in Lumbridge can revert/downgrade ironman status (7-day delay)

## Trade Restrictions (New Accounts)

- New accounts have GE trade restrictions on commonly botted items
- Removed after: 20+ hours playtime AND 100+ total skill level AND 10+ quest points
- Quick quest points: Sheep Shearer, Cook's Assistant, X Marks the Spot, Pirate's Treasure, Romeo & Juliet

## Starting in Lumbridge

### Adventure Paths

- Enabled if player told Gielinor Guide they're new
- Adventurer Jon stands near Lumbridge castle gates
- Provides Combat path starter kit
- Tasks: reach milestone levels (5, 10, 20) in combat/gathering skills for rewards
- Example reward: Level 10 Attack → Black Longsword

### Activity Adviser

- Accessed via orange orb with scroll icon below minimap
- Personalized recommendations based on skill levels, quest progress, recent activity
- Suggests content player is close to unlocking

### First Steps

- Deposit starter items in bank (top floor of Lumbridge Castle)
- Bank is pre-stocked with 25 gp
- Options: train combat (cows, chickens, goblins), train skills, start quests
- Recommended early quests: X Marks the Spot, Cook's Assistant

## Servers/Worlds

- 100+ game servers worldwide
- All worlds are identical except:
  - Physical hosting location (latency)
  - Silver star = F2P, Gold star = Members
  - Red star = dangerous PvP worlds
  - Black star = Deadman Mode
  - Green star = Leagues
  - Blue star / β = Unrestricted/Beta
  - Cyan star = Quest Speedrunning
- Some worlds have total level requirements
- Progress syncs across all worlds (except Unrestricted, Deadman, Leagues, Quest Speedrunning)

## Membership

- Members get 9 additional skills, larger game world, 100+ quests
- F2P has 15 skills and plenty of early-game content
- Membership only becomes fully advantageous at mid-game

## Account Security

- **Jagex Account**: Unified login with 2FA via email or authenticator app; store backup codes
- **RuneScape Authenticator**: Legacy system for accounts not yet on Jagex Account; 6-digit TOTP code every 30 seconds
- **Bank PIN**: Last line of defense; set up via any Banker
- Jagex NEVER contacts via anything except in-game Message Centre (exception: support portal tickets via email)

## Gameplay Mechanics

### Energy and Weight

- Finite run energy; depletes faster with more weight
- Weight = inventory + worn items (visible in Equipment Stats)
- More efficient to run when inventory is empty (e.g., running back to resource area after banking)

### Skills (F2P)

**Combat:** Attack, Strength, Defence, Ranged, Magic, Hitpoints, Prayer

| Skill | Function |
|-------|----------|
| Attack | Wield stronger melee weapons, melee accuracy |
| Strength | Melee max hit |
| Defence | Wear stronger armour, chance to block hits (does NOT reduce damage) |
| Ranged | Ranged accuracy and max hit |
| Magic | Magic accuracy, dodge magic, unlock spells |
| Hitpoints | Total HP pool |
| Prayer | Unlock prayers, prayer point pool |

**Gathering:** Mining, Fishing, Woodcutting

| Skill | Function |
|-------|----------|
| Mining | Gather ores; higher level = better pickaxes |
| Fishing | Catch fish at fishing spots |
| Woodcutting | Cut trees for logs; higher level = better axes |

**Artisan:** Runecraft, Crafting, Smithing, Cooking, Firemaking

| Skill | Function |
|-------|----------|
| Runecraft | Make runes from rune essence (Rune Mysteries quest recommended) |
| Crafting | Create items: jewellery, ranged armour |
| Smithing | Smelt ores → bars, smith bars → armour/weapons |
| Cooking | Cook food to heal Hitpoints |
| Firemaking | Make fires for cooking (alternative to ranges) |

### Combat System

- **Melee**: Max damage from Strength + equipment strength bonus; accuracy from Attack + equipment attack bonus
- **Ranged**: Both max damage and accuracy from Ranged level + ammo type + weapon
- **Magic**: Damage typically from spell, not Magic level; accuracy from Magic level + equipment

### Combat XP

- Normal: 4 XP per damage in the combat style used
- Hitpoints: 1.33 XP per 1 damage done
- Magic: base XP per spell cast + XP from damage (base XP granted even on splash)
- Defensive magic: 1 Magic XP + 1 Defence XP per damage + base casting XP
- Long range (Ranged): 2 Ranged XP + 2 Defence XP per damage
- Shared melee (e.g., Abyssal Whip): 1.33 XP each to Attack, Strength, Defence per damage

### Equipment Stats

- **Accuracy bonuses**: Chance to land a hit (per attack type)
- **Strength bonuses**: Increase max hit (all three combat schools)
- **Defence bonuses**: Chance to block (per attack type)
- **Prayer bonus**: Slows prayer point drain rate

### Death

- Occurs when Hitpoints reach 0
- Respawn at set respawn point
- Keep 3 most valuable items (by GE value, then alch value)
- Untradeable items also lost
- Remaining items go to gravestone at death location; fee to retrieve
- Different rules for instances, Wilderness, and PvP worlds

### Bank Notes

- "Withdraw as note" option at bank
- Noted items: tradeable to players, sellable to shops, weightless
- Cannot use or equip noted items
- Unnote by depositing in bank or using on a banker

## Grand Exchange

- Located NW Varrock
- Buy/sell any tradeable item via offer system
- Cannot view other players' offers
- Guide Price shown but may not reflect actual market price
- Buy offer: either instantly fills a sell offer (get change back) or waits
- Sell offer: either instantly fills a buy offer (get extra GP) or waits

## Early Game Activities

### Beginner (Lumbridge Area)

1. **Cut trees** — Woodcutting + Firemaking; switch to oaks at level 15; use best axe for WC level (not Attack level)
2. **Kill chickens** — Easy combat XP; bury bones; save feathers for fishing; two locations in Lumbridge
3. **Fish/cook shrimps** — Lumbridge Swamp; net fishing; drop for faster XP or bank for food stockpile
4. **Kill cows** — Combat XP + bones + raw beef + cowhides; cow pen north of castle or north of Al Kharid gates
5. **Tan cowhides** — Take to Al Kharid tanner; craft leather items (level 18 for leather chaps); buy needle + thread from Dommik's
6. **Fly fishing** — East side of river near Lumbridge furnace; need feathers + fly fishing rod; salmon at level 30
7. **Mine and smith** — Lumbridge mine; 14 tin + 14 copper → bronze bars at furnace → smith at anvil; use Home Teleport for furnace access
8. **Buy Chronicle** — From Diango in Draynor Village; 300 gp + 150 per teleport card; teleports south of Varrock (beware dark wizards)

### Intermediate

1. **Stronghold of Security** — Barbarian Village; rewards 10,000 gp + fighting/fancy boots; need ~15 Defence + food + armour; requires authenticator
2. **Buy wizard gear** — Varrock: staff of air from Zaff (4,400 gp total); blue wizard robe + hat; mind/fire rune packs from Aubury's
3. **Kill Lesser demon** — Wizards' Tower upper floor; caged = safe spot; need 33 Magic for Telekinetic Grab; drops Rune med helm
4. **Train on Al Kharid warriors** — Palace in Al Kharid; need ~15 in melee skills; close to bank and fishing

### Advanced

1. **Cut yew trees** — Level 60+ WC; rune axe; locations: south Edgeville bank, south Falador, east GE, west Rimmington, west Lumbridge Castle; Forestry worlds give +1 WC bonus per player (up to +10)
2. **Mining Guild** — Level 60+ Mining; rune pickaxe; iron, coal, mithril, adamantite rocks
3. **Kill Hill Giants** — Edgeville Dungeon; need Brass key (GE or navigate dungeon); big bones worth 329 gp; safespots available
4. **Dragon Slayer I** — Major F2P quest milestone

## PvP

### Methods

| Method | Safe Death | Notes |
|--------|-----------|-------|
| Emir's Arena | Yes | NE of Al Kharid; agreed duels |
| Clan Wars | Yes | Free-form PvP |
| Castle Wars | Yes | Minigame via Clan Wars portal |
| Wilderness | No | Any world; PK worlds most active |
| World 561/577 | No | F2P PvP-everywhere worlds (alternates weekly); safe zones exist |

### Skull Mechanic

- Attacker gets skull icon overhead
- Skulled player loses ALL items on death (Protect Item prayer keeps 1)
- Warns other players the skulled player is dangerous

### PvP Worlds for Non-PvP

- Can be used for uncrowded skilling/PvM spots
- Risk of being attacked anywhere except safe areas (banks)
- Always bring emergency teleport, food, minimal valuables
